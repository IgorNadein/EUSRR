# Быстрый старт: Настройка интеграции мессенджеров

## Что было добавлено:

✅ **3 новые модели базы данных:**
- `MessengerIntegration` - настройки подключения
- `MessengerAccount` - привязка аккаунтов сотрудников
- `MessengerMessage` - синхронизация сообщений

✅ **Админ-панель** для управления интеграциями

✅ **Страницы мессенджеров** с инструкциями

## Как начать:

### 1. Откройте админ-панель

Перейдите: http://localhost:9000/admin/communications/

Вы увидите новые разделы:
- **Интеграции с мессенджерами**
- **Аккаунты в мессенджерах**
- **Сообщения из мессенджеров**

### 2. Создайте интеграцию (пример для Telegram)

1. Нажмите "Интеграции с мессенджерами" → "Добавить"

2. Заполните:
   - **Тип мессенджера**: Telegram
   - **Включена**: ✓
   - **Статус**: Active
   - **API ключ**: (получите от @BotFather в Telegram)

3. Сохраните

### 3. Проверьте результат

Перейдите в раздел "Мои чаты":
- В боковой панели появились иконки мессенджеров
- Кликните на иконку Telegram
- Вы увидите страницу с инструкциями

Если вы администратор:
- Появится кнопка "Настроить интеграцию"
- Будут видны подробные инструкции

## Для администраторов

### Получение API ключей:

**Telegram (самый простой):**
1. Откройте @BotFather в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте Bot Token
5. Вставьте в админ-панель

**WhatsApp Business API:**
1. https://business.whatsapp.com/
2. Создайте приложение в Facebook for Developers
3. Получите Access Token

**WeChat:**
1. https://mp.weixin.qq.com/
2. Зарегистрируйте Official Account
3. Получите AppID и AppSecret

## Следующие шаги (для разработки)

Для полноценной работы интеграций потребуется реализовать:

1. **Webhook handlers** - обработка входящих сообщений
2. **Sync services** - синхронизация сообщений
3. **API endpoints** - для отправки сообщений
4. **Celery tasks** - фоновая синхронизация

Базовая инфраструктура готова, можно начинать интеграцию!
