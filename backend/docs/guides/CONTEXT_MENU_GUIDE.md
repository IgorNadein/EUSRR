# Контекстное меню сообщений

## ✅ Реализовано

Добавлено современное контекстное меню для сообщений (как в Telegram/WhatsApp/Viber).

---

## 🎯 Функционал

### Открытие меню

#### На десктопе (ПК)
- **Правый клик** на сообщении → открывается контекстное меню

#### На мобильных устройствах
- **Долгое нажатие** (500ms) на сообщении → вибрация → открывается контекстное меню
- Если палец двигается при нажатии → долгое нажатие отменяется

### Содержимое меню

#### 1. Быстрые реакции (верхняя секция)
```
👍  ❤️  😂  😮  😢  🙏  👏  🔥
```
- Одним тапом можно поставить реакцию
- Круглые кнопки с анимацией увеличения при нажатии

#### 2. Действия с сообщением (нижняя секция)

**Для всех сообщений:**
- 🔄 **Ответить** - цитировать сообщение
- 📋 **Копировать** - скопировать текст в буфер обмена
- ↗️ **Переслать** - переслать в другой чат

**Только для своих сообщений:**
- ✏️ **Редактировать** - изменить текст
- 🗑️ **Удалить** - удалить сообщение (красная кнопка)

### Закрытие меню

- Клик вне меню
- Нажатие клавиши **Escape**
- Скролл страницы
- После выбора любого действия

## 🎨 Дизайн

### Десктоп
- Округлое меню с тенью
- Позиционируется рядом с курсором
- Минимальная ширина 280px

### Мобильные
- Меню растягивается на всю ширину экрана (с отступами 16px)
- Закреплено внизу экрана
- Увеличенные кнопки (48x48px) для удобства тапов
- Анимация появления снизу вверх

### Эффекты
- Подсветка выбранного сообщения
- Плавные анимации (transform + opacity)
- Вибрация при долгом нажатии (мобильные)
- Toast уведомление "Текст скопирован"

### Тёмная тема
- Автоматическое определение через `prefers-color-scheme`
- Тёмный фон (#2b2d31)
- Светлый текст (#dbdee1)

## 🔧 API

### Инициализация
```javascript
const contextMenu = new MessageContextMenu({
    currentUserId: 123,
    emojis: ['👍', '❤️', '😂', ...],
    onReactionSelect: (messageId, emoji) => { ... },
    onEdit: (messageId) => { ... },
    onDelete: (messageId) => { ... },
    onReply: (messageId) => { ... },
    onForward: (messageId) => { ... }
});
```

### Подключение к сообщению
```javascript
const messageElement = document.querySelector('[data-message-id="123"]');
contextMenu.attachToMessage(messageElement);
```

### Методы
- `showMenu(element, messageId, isOwn, x, y)` - показать меню
- `closeMenu()` - закрыть меню
- `handleAction(action, messageId, emoji)` - обработать действие
- `showToast(message)` - показать уведомление

## 📱 UX особенности

### Предотвращение конфликтов
- Долгое нажатие отменяется если палец двигается (скролл)
- Клик по сообщению не открывает меню (только долгое нажатие)
- Меню автоматически закрывается при скролле

### Позиционирование
- Меню не выходит за границы экрана
- Автоматическое смещение если не помещается
- На мобильных всегда внизу экрана

### Тактильная обратная связь
- Вибрация 50ms при открытии меню (мобильные)
- Hover эффекты на кнопках (десктоп)
- Active состояния для тапов (мобильные)
