# План тестирования API заявлений (Requests)

## 🎯 Цель
Полное покрытие тестами логики доступа к заявлениям с учетом:
- Модельных прав (permissions)
- Департаментных прав (scoped permissions)
- Получателей (recipients/cc_users)
- Отделов (departments)
- Флага sent_to_all_department
