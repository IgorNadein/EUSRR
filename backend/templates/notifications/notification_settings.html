Email
Отправка уведомлений на вашу почту. Для включения отметьте чекбокс, затем выберите режим:
  • Мгновенно - каждое уведомление отправляется сразу
  • Ежедневный дайджест - все уведомления за день в одном письме
  • Еженедельный дайджест - все уведомления за неделю в одном письме
Чтобы отключить email, снимите галочку с "Email уведомления"
Telegram
Мгновенные уведомления в Telegram бот. Для настройки:
  1. Узнайте ваш Chat ID через бота @userinfobot
  2. Добавьте Chat ID в поле "Telegram Chat ID" в вашем профиле
  3. Отправьте /start нашему боту @{{ telegram_bot_username|default:"eusrr_bot" }}
  4. Включите галочку "Telegram" для нужных категорий
