# 📋 Краткий Summary: Исправление багов уведомлений

## 🔴 Проблема

Пользователи **не получали уведомления при согласовании заявлений** (одобрение/отклонение).

## 🔍 Причина найдена

**Коммит `c0b659f9` (29 декабря 11:02)** - ошибочно исключил автора заявления из получателей уведомлений с предположением:
> "Не уведомляем автора заявки (он видит результат на странице)"

**Но это было неверно:**
1. Коммит `935c12a` (29 декабря 11:19) убрал `alert()` с синхронным уведомлением
2. Значит, уведомление стало **единственным способом** информировать автора
3. Результат: автор вообще ничего не узнавал ❌

## ✅ Решение

**Коммит `de0caea` (5 января 2026)**

Удалено исключение автора из функции `notify_status_change()`:

```diff
- exclude_ids.add(request_obj.employee.id)  # ❌ Убрано!
+ # Автор теперь ВСЕГДА получает уведомление о решении
```
