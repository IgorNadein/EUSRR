# Документация проекта EUSRR

Эта папка содержит всю документацию проекта, организованную по категориям.

## 📁 Структура

### [completed/](completed/)
Завершенные задачи: рефакторинги, исправления, реализованные фичи (22 файла)

### [guides/](guides/)
Активные руководства и инструкции для разработчиков (гайды по валидации, миграции, настройке)

### [architecture/](architecture/)
Архитектурная документация и анализ систем (рендеринг чата, архитектура страниц)

### [diagnostic/](diagnostic/)
Диагностические руководства (WebSocket 1006, отладка проблем)

### [in-progress/](in-progress/)
Активные задачи в разработке (закупки, получатели заявок, календарь)

### [reports/](reports/)
Отчеты и аудиты проекта (реорганизация документации, план очистки)

- [README.md](../README.md) - основная документация проекта
