## 🎯 Workflow

1. **Планирование** - создается _PLAN.md
2. **Разработка** - ведется _PROGRESS.md
3. **Тестирование** - используется _TESTING.md
4. **Завершение** - файлы перемещаются в `completed/` с суффиксом _COMPLETE.md

## 📝 Примечание

При завершении задачи:
- Переименуйте документ в *_COMPLETE.md
- Переместите в `../completed/`
- Обновите changelog проекта
