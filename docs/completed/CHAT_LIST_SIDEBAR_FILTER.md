# Фильтрация чатов с боковой панелью типов

## Описание изменений

Страница списка чатов (`/communications/chats/`) была реорганизована с добавлением боковой панели для фильтрации по типам чатов.

## Структура

### Двухколоночный макет

1. **Левая колонка (боковая панель)** - фиксированная ширина (120px на desktop, адаптивно до 60px на mobile)
   - Кнопки-иконки для каждого типа чата
   - Бейджи с количеством чатов в каждой категории
   - Sticky-позиционирование (прилипает при скролле)

2. **Правая колонка (основной список)** - flex-grow для занятия оставшегося пространства
   - Все секции чатов (глобальный, отделы, личные, группы, каналы, объявления)
   - Поиск и существующие фильтры

## Типы чатов

Доступные типы для фильтрации:

- **Все** (`all`) - показывает все чаты
- **Глобальный** (`global`) - глобальный чат организации
- **Отделы** (`department`) - чаты отделов
- **Личные** (`private`) - личные чаты между пользователями
- **Группы** (`group`) - групповые чаты
- **Каналы** (`channel`) - каналы для широковещательных сообщений
- **Объявления** (`announcement`) - каналы объявлений

## Функциональность

### Фильтрация

- Клик по кнопке типа чата показывает только чаты этого типа
- Кнопка "Все" показывает все чаты
- Активная кнопка подсвечивается синим цветом
- При наведении иконка увеличивается (scale 1.1)

### Счетчики

- Каждая кнопка показывает количество чатов в категории
- Счетчики обновляются динамически при поиске
- Пустые категории скрывают бейдж (display: none)

### Адаптивность

**Desktop (>992px):**
- Ширина панели: 120px
- Отображаются иконки + текст + бейджи

**Tablet (768-992px):**
- Ширина панели: 90px
- Уменьшенный шрифт текста

**Mobile (576-768px):**
- Ширина панели: 70px
- Еще более компактный вид

**Small Mobile (<576px):**
- Ширина панели: 60px
- Скрывается текст, остаются только иконки
- Только title атрибут для понимания назначения

## Интеграция с существующими функциями

### Поиск
- Поиск работает в рамках отфильтрованных секций
- Счетчики обновляются после применения поиска
- При смене фильтра поиск переприменяется

### WebSocket
- Обновления чатов в реальном времени сохраняются
- Счетчики могут быть обновлены при получении новых сообщений

### Создание чата
- Модальное окно создания чата не изменено
- После создания нового чата рекомендуется добавить обновление счетчиков
