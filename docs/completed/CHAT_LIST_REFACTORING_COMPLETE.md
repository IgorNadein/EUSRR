## 📦 Созданные компоненты

### 1. `includes/components/chat_avatar.html`
**Назначение:** Аватар для карточки чата

**Функциональность:**
- Отображает аватар для всех типов чатов
- global → иконка глобуса или аватар чата
- department → иконка отдела или аватар
- private → аватар собеседника
- group/channel/announcement → аватар или иконка

---

### 2. `includes/components/chat_title.html`
**Назначение:** Заголовок карточки чата

**Функциональность:**
- Название чата или имена участников (для private)
- Бейдж "Основной" для главных чатов
- Счётчик сообщений
- Счётчик непрочитанных

---

### 3. `includes/components/chat_last_message.html`
**Назначение:** Последнее сообщение в чате

**Функциональность:**
- Для department: показывает название отдела
- Время последнего сообщения
- Автор последнего сообщения
- Превью контента (120 символов)
- Заглушка "Сообщений пока нет"

---

### 4. `includes/components/chat_menu.html`
**Назначение:** Меню действий для чата

**Функциональность:**
- Закрепить/открепить
- Настройки уведомлений
- Редактировать (кроме основных)
- Удалить (кроме основных)

---

### 5. `includes/components/chat_card.html`
**Назначение:** Главный компонент - полная карточка чата

**Функциональность:**
- Объединяет все подкомпоненты
- Единый шаблон для всех типов чатов
- Data-атрибуты для JS (chat-id, type, last-ts, haystack)

```django

  {% endfor %}
</div>
```
